# Skyfall Ledger

Ledger is the FinTech UI kit in the Skyfall family of design systems. It provides a comprehensive set of React components purpose-built for financial SaaS products — from banking dashboards and payment flows to transaction management and operational tools.

## What Ledger is for

Ledger is designed to accelerate product design and front-end development for financial applications, including:

- **Banking and payments interfaces** — accounts, transfers, payment processing
- **Transaction management** — ledgers, statements, audit trails
- **Financial dashboards** — portfolio views, balance summaries, KPIs
- **Data-dense operational tools** — tables, filters, bulk actions
- **Compliance and admin workflows** — forms, approvals, status tracking

## Who Ledger is for

Ledger is built for product engineers, designers, and teams building financial products that need to be precise, trustworthy, and production-ready from day one.

## System structure

Ledger is organized into two layers:

### Components

The shared core system — foundational building blocks used across all Skyfall UI kits. This includes inputs, data display, feedback, surfaces, navigation, layout primitives, and utilities. These components follow atomic design principles and compose predictably.

**Current core coverage:**

| **Surfaces** | 6 | Modal, Drawer, Accordion, Card, Paper, Popover |

| **Utils** | 5 | Portal, VisuallyHidden, Backdrop, ClickAwayListener, NoSSR |

### Ledger X

The FinTech-specific extension layer. This includes domain components tailored to financial workflows and a Recharts-based data visualization system designed for financial dashboards and reports.

## Design principles

Ledger components are built around four qualities:

- **Precision** — every pixel serves the data. Tabular numerics, aligned columns, and measured whitespace
- **Trust** — consistent visual language and predictable behavior that instills confidence in financial contexts
- **Clarity** — clean information hierarchy that supports data-dense interfaces without visual noise
- **Composability** — atomic primitives that compose into complex financial workflows
